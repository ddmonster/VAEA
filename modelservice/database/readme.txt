# 产品特性说明：
# 1. urls 产品网站
# 2. product_name 产品名
# 3. brand 品牌
# 4. star_rating 星级
# 5. number_of_rating 评论数量
# 6. mrp 市场价或者建议零售价（单位₹ 印度卢比）
# 7. sale_price 售价（单位₹ 印度卢比）
# 8. colour 颜色
# 9. capacity 容量（带单位）
# 10. wattage 瓦特功耗（不带单位）
# 11. country_of_origin 原产国
# 12. home_kitchen_rank 家用厨房用品排名
# 13. air_fryers_rank 空气炸锅排名
# 14. technical_details 技术细节，json数据，内含多种特性，但每个产品的这个json所包含的键并不一致，可以用于前端展示
# 15. description 自然语言的产品描述
# 16. asin 产品编号，和数据库中的ID列一样
# 17. capacity_1 容量不带单位
# 18. weight 重量（带单位）
# 19. Has_Nonstick_Coating 是否有不粘锅涂层
# 20. Naterial 材料
# 21. Manufacturer 制造商
# 22. Control_Method 控制技术（触摸或者app控制）
# 23. Model_Name 型号名称
# 24. Recommended_Uses_For_Product 产品用途（烤、炙、蒸、加热）
# 25. Special Feature 特殊功能
# 26. Max_Temperature_Setting 最高温度设置
# 27. Imported_By 进口商地址
